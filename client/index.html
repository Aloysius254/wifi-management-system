How Client Isolation Works

Each room is assigned a Virtual Access Point (VAP) with a unique VLAN ID. Devices connected through different VAPs cannot communicate with each other — they cannot see printers, file shares, or other devices from other rooms. This replaces the shared WPA2 password model where all guests were on the same network segment.

When a guest uses excessive bandwidth (e.g. BitTorrent), the system detects usage above the throttle threshold and flags the session. In a production deployment, this flag would trigger QoS rules on the router to limit that session's bandwidth automatically.
